# Security Policy

## Supported Versions

The current `main` branch is supported.

## Reporting a Vulnerability

Please open a private security advisory on GitHub if the repository has advisories enabled.

If advisories are not enabled, open an issue with a minimal description and avoid posting exploit details publicly. A maintainer can then coordinate a private channel.

## Permission Model

Timestamp Inserter requires macOS Accessibility permission to send text input events to the focused app. The app does not read active window contents, collect data, or make network requests.
