# Code of Conduct

This project follows the Contributor Covenant Code of Conduct.

## Our Standards

Examples of behavior that contributes to a positive environment:

- Being respectful of differing viewpoints and experience levels.
- Giving and accepting constructive feedback.
- Focusing on what is best for the project.

Examples of unacceptable behavior:

- Harassment, insults, or personal attacks.
- Public or private intimidation.
- Publishing private information without permission.

## Enforcement

Maintainers may remove, edit, or reject comments, commits, issues, and pull requests that do not align with this Code of Conduct.
